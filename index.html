Configure o Hub para começar

O Hub usa o GitHub como banco de dados gratuito. Um token é necessário para ler e gravar a lista de orçamentos. A configuração é feita uma única vez.

1. Acesse github.com → Settings → Developer settings → Personal access tokens (classic)
2. Marque apenas o escopo gist
3. Clique em "Generate token", copie e cole abaixo

Hospedagem gratuita: Crie um repositório no GitHub, suba os dois arquivos (index.html e orcamento-veli-v5.html) e ative GitHub Pages nas configurações. Seu hub ficará em seuusuario.github.io/nome-do-repo/ sem custo adicional.

O token é guardado apenas no seu navegador (localStorage) e nunca é enviado a nenhum servidor além do GitHub.
